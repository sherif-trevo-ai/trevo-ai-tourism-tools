# System Architecture

This project is evolving into a lightweight SaaS-style system for tourism businesses.

---

## Components

### 1. Frontend Layer

* Simple web interface
* Collects user input
* Sends requests to API

---

### 2. API Layer (Backend)

Built using Flask.

Endpoints:

* /roi → calculates ROI
* /marketing → generates marketing messages

---

### 3. Tools Engine

Core logic:

* ROI Calculator
* Marketing Generator
* Future AI tools

---

## Flow

User → Web UI → API → Tools Engine → Response

---

## Future Expansion

* Authentication system
* Database integration
* AI-powered generation
* Dashboard analytics
* Multi-tenant SaaS model
